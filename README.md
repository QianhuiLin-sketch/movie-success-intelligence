# Movie Success Intelligence

## Project Overview
This project analyses movie box office drivers and rating patterns using Python. It is designed as a small decision-support product for film investors, producers, and studio analysts who want to understand which factors may be associated with movie success.

## Analytical Problem
This project investigates:
- which pre-release factors may help support movie investment decisions
- which post-release variables help explain box office performance

## Target User
The target users are film investors, producers, and studio analysts.

## Dataset
The dataset used in this project is the IMDb 5000 Movie Dataset downloaded from Kaggle.

Source: Kaggle  
Access date: 17 April 2026

## Methods
The notebook includes:
- data cleaning and preparation
- exploratory data analysis
- feature engineering
- pre-release modelling
- model evaluation
- feature importance analysis
- post-release explanatory analysis

## Repository Structure
- `ACC102-2_.ipynb`: main notebook
- `movie_metadata.csv`: dataset
- `README.md`: project description

## How to Run
1. Place the dataset file in the same folder as the notebook.
2. Open the notebook in Jupyter Notebook or JupyterLab.
3. Run all cells from top to bottom.

## Limitations
This project has several limitations. The dataset may not represent the entire movie market, some variables contain missing values, and the model results should be interpreted as educational analysis rather than professional forecasting.
